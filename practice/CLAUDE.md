# MOP 프로젝트

MOP(Marketing Optimization Platform)는 LG CNS가 제공하는 AI 광고 최적화 서비스다.
광고주와 대행사가 검색광고·쇼핑검색광고를 자동으로 최적화할 수 있도록 돕는다.

## 서비스 핵심 개념

- **CTR (클릭률)**: 광고가 노출됐을 때 실제로 클릭한 비율. 높을수록 좋음.
- **입찰가**: 광고가 상위에 노출되기 위해 지불하는 금액. MOP가 자동 조정함.
- **캠페인**: 광고의 묶음 단위. 예산과 목표를 설정하는 기본 단위.
- **소재**: 실제로 노출되는 광고 문구와 이미지.
- **ROAS**: 광고비 대비 매출. 100만원 광고비로 500만원 매출 = ROAS 500%.

## 주요 사용자

- **광고주 담당자**: 자사 광고를 직접 운영하는 마케터
- **대행사 AE**: 여러 클라이언트 광고를 동시에 관리하는 담당자

## 지원 문서 위치

- 사용 가이드: support.mop.co.kr
- 주요 기능: 입찰 최적화 / 예산 배분 / 성과 리포트 / 이상 감지 알림

## 작업 시 주의사항

- 광고 용어는 한국어로 설명할 것 (CTR, ROAS 등은 영어 약자 그대로 사용하되 뜻은 병기)
- 사용자는 마케터이므로 개발 용어 사용 금지
- 수치 예시를 들 때는 실제 광고 업계 기준에 맞출 것
- 답변은 항상 3줄 이내로 간결하게 할 것
