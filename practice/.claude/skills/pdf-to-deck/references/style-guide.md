# Deck Style Guide

발표 슬라이드에 적용할 디자인 규칙. `build_deck.py`가 이미 반영하고 있으나,
변형이 필요할 때 이 문서를 참고해 스크립트를 수정하라.

## 비율 / 여백
- 16:9 (13.33 x 7.5 inch)
- 좌측 여백 0.6", 본문 시작 0.95"

## 컬러
- Accent: `#1F6FEB` (파란색 사이드바, 강조)
- Ink: `#1F2328` (제목·본문)
- Muted: `#6A737D` (부제·캡션)

## 타이포그래피
- 제목 32pt Bold
- 본문 22pt Regular
- 부제 20pt Muted
- 시스템 기본 폰트 사용 (한글이면 `맑은 고딕` 또는 `Apple SD Gothic Neo`)

## 레이아웃 원칙
- **한 슬라이드 한 메시지** — 제목이 그대로 결론이 되도록 작성
- bullet 최대 3개, 각 30자 이내
- 숫자/비교는 텍스트보다 표 한 줄을 선호
- 발표자 노트(speaker notes)에 1분짜리 구두 설명 작성
