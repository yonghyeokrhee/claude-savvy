# 쇼핑검색광고 심층 분석 보고서

> 기준 기간: 2026년 2월 | 작성일: 2026-03-16 | 분석 도구: MOP

---

## 한 줄 진단

쇼핑검색광고는 ROAS 698%로 검색광고(206%) 대비 3.4배 높은 효율을 기록 중이며, 현재 예산 배분(전체의 25%)은 명백하게 과소 투자 상태입니다.

---

## 캠페인별 성과 카드

### 브랜드_쇼핑검색

| 지표 | 수치 |
|---|---|
| ROAS | 750% |
| CTR | 5.0% |
| CPC | 400원 |
| 광고비 | 1,700,000원 |
| 매출 | 12,750,000원 |

**등급: 우수 (전 지표 최상위)**

**진단**
자사 브랜드 키워드에 상품 이미지와 가격이 직접 노출되는 쇼핑검색의 구조적 강점이 그대로 수치로 나타납니다. 브랜드를 이미 인지한 고객이 상품을 탐색하는 단계에서 구매 의도가 높기 때문에 전환율이 높습니다. CTR 5.0%는 쇼핑검색 전체 평균(4.7%)을 상회하며, CPC 400원은 평가 기준 우수 구간(400원 이하)의 경계에 정확히 위치합니다.

**액션**
- 현재 예산 1,700,000원을 3월에 2,500,000원 이상으로 단계적 확대
- 목표입찰 ROAS를 700%로 설정하여 MOP 자동 최적화 유지
- 상위 클릭 상품군의 소재(이미지, 가격 표기) 개선으로 CTR 5.5% 목표 설정
- 품절 이상 감지 기능 활성화하여 재고 부족 상품이 광고 낭비 없도록 관리

---

### 리타겟팅_쇼핑

| 지표 | 수치 |
|---|---|
| ROAS | 600% |
| CTR | 4.0% |
| CPC | 500원 |
| 광고비 | 900,000원 |
| 매출 | 5,400,000원 |

**등급: 우수**

**진단**
사이트 방문 후 구매하지 않은 고객에게 재노출하는 전략으로, ROAS 600%는 우수 기준(300% 이상)을 두 배 초과합니다. 브랜드_쇼핑검색 대비 ROAS가 150%p 낮은 이유는 리타겟팅 대상의 구매 의도가 초기 관심 단계에 머무는 비율이 포함되기 때문으로 자연스러운 수치입니다. CTR 4.0%는 평가 기준 우수 구간의 하한선으로, 소재 다양화를 통해 추가 개선 여지가 있습니다.

**액션**
- 현 전략 기조 유지하면서 예산을 900,000원 → 1,400,000원으로 확대
- 장바구니 이탈 고객과 상품 조회 고객의 소재를 분리하여 맞춤 노출 전환
- 리타겟팅 기간(윈도우)을 7일 / 14일 / 30일로 세분화하여 기간별 ROAS 비교
- 경쟁 심화도 분석을 통해 입찰 과열 시간대 조정

## 브랜드 vs 리타겟팅 전략 분석

### 두 전략의 역할 차이

| 구분 | 브랜드_쇼핑검색 | 리타겟팅_쇼핑 |
|---|---|---|
| 타겟 | 브랜드 인지 고객 (신규 포함) | 방문 후 미구매 고객 |
| 구매 단계 | 탐색 → 구매 결정 | 이탈 → 재유입 |
| ROAS | 750% | 600% |
| CTR | 5.0% | 4.0% |
| CPC | 400원 | 500원 |
| 예산 비중 | 65% (1,700,000원) | 35% (900,000원) |

### 현재 전략의 강점과 개선 포인트

**강점.** 두 캠페인 모두 ROAS 300% 기준을 압도적으로 초과하여 포트폴리오 전체가 건강합니다. 브랜드 캠페인이 퍼널 상단과 하단을 동시에 커버하고, 리타겟팅이 이탈 고객을 회수하는 이중 구조가 효과적으로 작동하고 있습니다.

**최적 비중 제안.** 현재 65:35 비율은 브랜드 캠페인의 압도적 효율을 감안할 때 합리적입니다. 다만 예산 확대 시 브랜드 캠페인에 추가 자원을 먼저 집중(70:30 → 이후 데이터 기반 재조정)하는 것을 권장합니다. 리타겟팅은 방문자 모수가 늘어야 효과가 커지므로 브랜드 캠페인 확대 → 방문자 증가 → 리타겟팅 확대의 선순환 구조를 활용합니다.

## 검색광고 대비 효율성

### 핵심 지표 비교

| 지표 | 쇼핑검색광고 | 검색광고 | 쇼핑검색 우위 |
|---|---|---|---|
| ROAS | 698% | 206% | +492%p (3.4배) |
| CTR | 4.7% | 1.5% | +3.2%p (3.1배) |
| CPC | 430원 | 1,139원 | 62% 저렴 |
| 예산 비중 | 25% (2,600,000원) | 75% (8,200,000원) | — |
| 매출 기여 | 51.8% (18,150,000원) | 48.2% (16,900,000원) | 예산 대비 역전 |

### 동일 예산 투입 시 매출 비교

쇼핑검색과 검색광고에 동일하게 2,600,000원을 투입했을 때의 결과:

- 검색광고: 2,600,000원 × 206% = **5,356,000원 매출**
- 쇼핑검색: 2,600,000원 × 698% = **18,148,000원 매출**
- 차이: 동일 예산으로 **12,792,000원 추가 매출** 창출 가능

### 채널별 클릭 1회당 매출

- 검색광고: 16,900,000원 / 7,200클릭 = **클릭당 2,347원 매출**
- 쇼핑검색: 18,150,000원 / 6,050클릭 = **클릭당 3,000원 매출**
- 쇼핑검색이 클릭 1회당 653원(28%) 더 많은 매출 창출

### 주의: 검색광고 내 캠페인별 격차

검색광고 전체 ROAS가 낮은 주된 원인은 특정 캠페인의 부진입니다.

| 캠페인 | ROAS | 평가 |
|---|---|---|
| 봄신상_검색광고 | 400% | 우수 |
| 경쟁사키워드_검색 | 150% | 개선 필요 (기준 미달) |
| 신규고객_검색 | 100% | 개선 필요 (손익분기 수준) |

경쟁사키워드와 신규고객 캠페인에서 집행 중인 5,800,000원(검색광고 예산의 71%)이 ROAS 100~150% 수준에 머물고 있어, 이 예산의 일부를 쇼핑검색으로 전환하면 즉시 효율 개선이 가능합니다.

## 확대 투자 시나리오

### 전제 조건

- 3월 총예산: 12,000,000원 (2월과 동일 가정)
- 현재 쇼핑검색 예산 비중: 25% (3,000,000원 배정, 2,600,000원 집행)
- 제안 시나리오: 쇼핑검색 비중을 40%로 확대

### 시나리오 비교

| 구분 | 현재 (2월) | 보수적 시나리오 | 낙관적 시나리오 |
|---|---|---|---|
| 쇼핑검색 예산 | 2,600,000원 | 4,800,000원 | 4,800,000원 |
| 예상 ROAS | 698% (실적) | 600% (하향 가정) | 698% (유지 가정) |
| 예상 매출 | 18,150,000원 | 28,800,000원 | 33,504,000원 |
| 매출 증가 | 기준 | +10,650,000원 | +15,354,000원 |

**보수적 시나리오 근거.** 예산 확대 시 입찰 경쟁이 강화되어 CPC가 오를 수 있고, 상위 노출 자리 수 제한으로 인해 전환율이 소폭 하락할 가능성을 반영하여 ROAS를 현재 대비 98%p 낮게 설정했습니다.

**낙관적 시나리오 근거.** MOP 목표입찰 기능이 ROAS 효율을 유지하는 방향으로 자동 입찰을 조정하고, 봄 시즌 수요 증가로 전환율이 현 수준을 유지하는 경우입니다.

### 재원 마련 방안 (추가 예산 없이 실행)

쇼핑검색 확대를 위한 예산을 검색광고 비효율 캠페인에서 확보합니다.

- 경쟁사키워드_검색: 3,000,000원 → 1,500,000원으로 50% 절감 (ROAS 150%, 기준 미달)
- 신규고객_검색: 2,800,000원 → 1,500,000원으로 46% 절감 (ROAS 100%, 손익분기)
- 절감 재원 합계: 2,800,000원
- 해당 재원을 쇼핑검색에 전환 시 총 쇼핑검색 예산: 5,400,000원 (전체의 45%)

이 경우 보수적 ROAS(600%) 기준으로도 **32,400,000원 매출** 달성이 가능하며, 이는 현재 쇼핑검색 매출 대비 78.5% 증가입니다.
