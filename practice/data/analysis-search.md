# 검색광고 심층 분석 보고서

## 한 줄 진단

검색광고 전체 ROAS 206%로 간신히 손익 방어 수준이며, 3개 캠페인 중 2개가 기준 미달로 예산 대부분이 비효율 구간에 묶여 있는 상태입니다.

## 캠페인별 성과 카드

### 봄신상_검색광고

| 지표 | 수치 | 평가 |
|---|---|---|
| ROAS | 400% | 우수 |
| CTR | 3.0% | 우수 |
| 클릭수 | 3,600회 | - |
| 광고비 | 2,400,000원 | - |
| 매출 | 9,600,000원 | - |

- 등급: **우수**
- 진단: 봄 시즌 키워드에 대한 선제적 입찰가 인상이 CTR 3.0%와 ROAS 400% 달성으로 이어졌습니다. 검색광고 3개 캠페인 중 유일하게 우수 등급을 기록했으며, 전체 검색광고 매출의 56.8%(9,600,000원 / 16,900,000원)를 이 캠페인 혼자 창출하고 있습니다. 광고비는 전체 검색광고 예산의 29.3%만 사용하면서 절반 이상의 매출을 담당하는 핵심 캠페인입니다.
- 액션: 3월에도 봄 시즌 수요가 이어지는 만큼 예산을 현재 대비 30~50% 증액(약 3,000,000~3,600,000원)하고, 성과가 좋은 키워드 중심으로 확장 매칭 또는 유사 키워드를 추가 발굴해 노출 볼륨을 늘리는 것을 권장합니다.

### 경쟁사키워드_검색

| 지표 | 수치 | 평가 |
|---|---|---|
| ROAS | 150% | 개선 필요 |
| CTR | 1.0% | 개선 필요 |
| 클릭수 | 2,000회 | - |
| 광고비 | 3,000,000원 | - |
| 매출 | 4,500,000원 | - |

- 등급: **개선 필요**
- 진단: 경쟁사 브랜드 키워드를 타깃으로 하는 캠페인이지만, ROAS 150%로 광고비 대비 매출이 손실 구간에 있습니다. 경쟁사 키워드 입찰 경쟁이 치열해 CPC(클릭당 비용)가 높게 형성되는 반면, 경쟁사를 검색하는 사용자의 구매 전환율은 자사 브랜드 검색 대비 낮기 때문입니다. CTR 1.0%는 3개 캠페인 중 최하위권으로, 소재 메시지가 경쟁사 검색 의도와 맞지 않는 것으로 보입니다. 현재 3,000,000원이라는 높은 예산이 가장 낮은 효율을 만들고 있는 구조입니다.
- 액션: 즉시 예산을 1,500,000원 수준으로 50% 절감하고, 소재를 "경쟁사 대비 자사 강점 비교" 메시지로 전면 교체합니다. 2주 테스트 후 ROAS 200% 미달 시 캠페인 일시 중지 후 전략 재수립을 검토합니다. 아울러 MOP 제외키워드 관리 기능을 활용해 전환 가능성이 낮은 순수 경쟁사 브랜드 키워드를 제외 처리하는 작업이 필요합니다.

### 신규고객_검색

| 지표 | 수치 | 평가 |
|---|---|---|
| ROAS | 100% | 개선 필요 |
| CTR | 1.0% | 개선 필요 |
| 클릭수 | 1,600회 | - |
| 광고비 | 2,800,000원 | - |
| 매출 | 2,800,000원 | - |

- 등급: **개선 필요 (긴급)**
- 진단: ROAS 100%는 광고비와 매출이 동일한 완전한 손익분기점 상태를 의미합니다. 실질적으로는 운영 인건비, 물류비 등 부가 비용을 감안하면 이 캠페인을 운영할수록 적자가 발생하는 구조입니다. 신규고객 확보를 목표로 설정했다 하더라도, 2,800,000원을 투입해 같은 금액의 매출만 회수하는 것은 장기적 투자 관점으로도 정당화하기 어렵습니다. CTR 1.0%는 소재와 키워드 조합이 잠재 고객의 관심을 끌지 못하고 있음을 보여줍니다.
- 액션: 3월 예산을 최소화(500,000원 이하)하거나 일시 중지하고, 신규고객 확보 목표는 ROAS 698%를 기록 중인 쇼핑검색광고 리타겟팅으로 대체하는 것을 권장합니다. 만약 신규고객 캠페인을 유지하고자 한다면, 광범위한 일반 키워드 대신 구매 의도가 명확한 '브랜드명+구매/할인' 형태의 구체적 키워드로 전환하고 소재도 신규고객 전용 혜택(첫 구매 할인 등)을 전면에 내세우는 방향으로 개편해야 합니다.

## 채널 비교 분석

### 검색광고 vs 쇼핑검색광고 효율성 비교

| 비교 항목 | 검색광고 | 쇼핑검색광고 | 격차 |
|---|---|---|---|
| 총 노출수 | 480,000회 | 130,000회 | 검색광고 3.7배 많음 |
| 총 클릭수 | 7,200회 | 6,050회 | 검색광고 1.2배 많음 |
| 평균 CTR | 1.5% | 4.7% | 쇼핑검색 3.1배 높음 |
| 총 광고비 | 8,200,000원 | 2,600,000원 | 검색광고 3.2배 많음 |
| 총 매출 | 16,900,000원 | 18,150,000원 | 쇼핑검색이 더 높음 |
| 평균 ROAS | 206% | 698% | 쇼핑검색 3.4배 높음 |
| 예산 비중 | 75.9% | 24.1% | - |
| 매출 기여 비중 | 48.2% | 51.8% | - |

### 채널 진단 요약

검색광고는 전체 예산의 75.9%를 차지하면서 매출 기여는 48.2%에 그치고 있습니다. 반면 쇼핑검색광고는 예산 24.1%로 매출 51.8%를 창출하며 압도적 효율을 보이고 있습니다.

특히 CTR 격차가 두드러집니다. 쇼핑검색광고의 CTR 4.7%는 검색광고 1.5%의 3배 이상으로, 쇼핑검색의 상품 이미지 노출 방식이 클릭 유도에서 훨씬 유리한 포맷임을 보여줍니다. 노출 볼륨은 검색광고가 3.7배 많지만, 실제 클릭으로 연결되는 비율은 쇼핑검색이 월등히 높습니다.

검색광고가 가진 유일한 장점은 넓은 노출 볼륨(상단 텍스트 광고로 브랜드 인지도 확보)이지만, 현재 비용 구조로는 인지도 투자 대비 매출 회수가 이루어지지 않는 상황입니다.

## 키워드 전략 제안

### 즉시 실행 — 제외 키워드 정리

MOP 제외키워드 관리 기능을 활용해 아래 유형의 키워드를 즉시 제외 처리합니다.

- 경쟁사키워드_검색 캠페인: 순수 경쟁사 브랜드명 단독 키워드 (예: "[경쟁사명]", "[경쟁사명] 공식")
- 신규고객_검색 캠페인: 구매 의도 없는 정보성 키워드 (예: "후기", "리뷰", "비교", "어떤가요")

### 단기 — 봄신상 캠페인 키워드 확장

현재 봄신상_검색광고의 CTR 3.0%는 키워드-소재 조합이 잘 맞는다는 신호입니다. 아래 방향으로 확장을 검토합니다.

- 기존 성과 키워드의 확장 매칭 적용 (정확한 키워드만 운영 중이라면)
- "2026 봄", "신학기", "봄맞이", "봄 코디" 등 시즌 연관 키워드 추가
- 상위 클릭 키워드 기반 유사 키워드 발굴 (MOP 경쟁 분석 기능 활용)

### 중기 — 신규고객 캠페인 키워드 전략 전환

광범위한 신규고객 키워드 대신 구매 퍼널 하단의 고의도 키워드로 전환합니다.

- 전환 전: 일반 카테고리 키워드 (노출 많지만 전환율 낮음)
- 전환 후: "[브랜드명] 구매", "[브랜드명] 할인", "[상품명] 최저가" 등 구매 의도 명확한 키워드
- MOP 목표입찰 기능을 활용해 전환 중심 자동 입찰 적용 권장

### 입찰 전략 조정

| 캠페인 | 현재 상황 | 권장 입찰 전략 |
|---|---|---|
| 봄신상_검색광고 | 성과 우수 | 입찰가 유지 또는 소폭 인상, MOP 목표입찰 ROAS 350% 설정 |
| 경쟁사키워드_검색 | ROAS 150% 적자 | 입찰가 30~40% 인하, 상위 노출 포기하고 효율 집중 |
| 신규고객_검색 | ROAS 100% 손익분기 | 일시 중지 후 키워드 재구성, MOP Spend Pacing 활용해 일 예산 상한 설정 |

## 예산 재배분 의견

### 현재 문제점

전체 검색광고 예산 9,000,000원의 64.6%(경쟁사키워드 3,000,000원 + 신규고객 2,800,000원 = 5,800,000원)가 ROAS 200% 미만 캠페인에 집중되어 있습니다. 유일하게 성과가 좋은 봄신상 캠페인에는 26.7%(2,400,000원)만 배정된 역설적인 구조입니다.

### 3월 검색광고 내 예산 재배분 제안

| 캠페인 | 2월 예산 | 3월 제안 예산 | 변경 내용 |
|---|---|---|---|
| 봄신상_검색광고 | 2,400,000원 | 3,600,000원 | +1,200,000원 증액 (봄 시즌 피크 활용) |
| 경쟁사키워드_검색 | 3,000,000원 | 1,500,000원 | -1,500,000원 절감 (효율 회복 테스트 후 재평가) |
| 신규고객_검색 | 2,800,000원 | 500,000원 | -2,300,000원 절감 (최소 테스트 예산만 유지) |
| **검색광고 합계** | **8,200,000원** | **5,600,000원** | **-2,600,000원 절감** |

### 절감 예산의 쇼핑검색 이관 제안

검색광고에서 절감한 2,600,000원을 쇼핑검색광고로 이관하면 현재 ROAS 698% 기준으로 약 18,148,000원의 추가 매출을 기대할 수 있습니다.

| 채널 | 2월 예산 | 3월 제안 예산 | 변경 내용 |
|---|---|---|---|
| 검색광고 | 9,000,000원 | 6,600,000원 | -2,400,000원 |
| 쇼핑검색 | 3,000,000원 | 5,400,000원 | +2,400,000원 |
| **합계** | **12,000,000원** | **12,000,000원** | 동일 예산 유지 |

총 예산을 늘리지 않고도 쇼핑검색 비중을 현재 25%에서 45%로 높여 전체 ROAS 개선을 기대할 수 있는 전략입니다. 이는 budget-usage.md에서 제안한 "쇼핑검색 비중 40% 확대" 방향과도 일치합니다.
