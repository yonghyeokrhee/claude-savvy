# Claude Code 소개

## Claude Code란?

Anthropic이 공식 제공하는 터미널 기반 AI 코딩 에이전트. 채팅 창이 아닌 **코드베이스 안에서** 직접 동작한다.

| 구분 | 채팅형 AI | Claude Code |
|---|---|---|
| 코드 접근 | 복사·붙여넣기 | 파일 직접 읽기/쓰기 |
| 명령 실행 | 불가 | 셸 명령 실행 |
| 프로젝트 맥락 | 매번 설명 필요 | CLAUDE.md로 영속 유지 |

## 에이전트로서의 Claude Code

단순 코드 생성기가 아니라 **에이전트**다. 목표를 받아 스스로 계획하고, 도구를 선택해 실행하며, 결과를 확인하고 완료까지 자율적으로 동작한다.

```
사용자 요청 → 계획 수립 → 도구 실행 → 결과 확인 → 완료 or 재시도
```

## 설치 및 시작

```bash
npm install -g @anthropic-ai/claude-code
cd my-project
claude
```

## 핵심 도구

| 도구 | 역할 |
|---|---|
| Read / Write / Edit | 파일 읽기·생성·수정 |
| Bash | 셸 명령 실행 |
| Glob / Grep | 파일·내용 검색 |
| WebSearch / WebFetch | 웹 검색 및 URL 조회 |

## CLAUDE.md — 프로젝트 지침서

프로젝트 루트에 두면 매 세션 시작 시 자동으로 읽힌다. 기술 스택, 컨벤션, 중요 파일 경로를 정의해두면 매번 설명할 필요가 없다.

## 실전 팁

- **구체적으로 요청** — 파일명, 함수명, 제약 조건을 함께 명시
- **단계별 진행** — 큰 작업은 나눠서 체크포인트마다 확인
- **Git 안전망** — 대규모 변경 전 `git commit`으로 되돌릴 지점 확보
- **`/clear`** — 세션이 길어져 컨텍스트가 오염되면 초기화
- **`/cost`** — 토큰 사용량 수시 확인
