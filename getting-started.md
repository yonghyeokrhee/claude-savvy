# 시작하기

## 설치

```bash
npm install -g @anthropic-ai/claude-code
```

## 인증

처음 실행하면 Anthropic 계정 로그인 화면으로 이동합니다:

```bash
claude
```

## 세션 시작

```bash
cd my-project
claude
```

한 줄 작업도 가능합니다:

```bash
claude "이 프로젝트가 뭐 하는 건지 설명해줘"
```

## 주요 명령어

| 명령어 | 설명 |
|---|---|
| `claude` | 대화형 세션 시작 |
| `/help` | 세션 내 도움말 |
| `/clear` | 컨텍스트 초기화 |
| `/cost` | 토큰 사용량 확인 |
| `/exit` | 세션 종료 |
