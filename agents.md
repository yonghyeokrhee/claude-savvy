# Agent 도구

## Agent 도구란?

Claude Code에는 `Agent`라는 특수한 내장 도구가 있습니다. 이 도구는 **독립적인 Claude 인스턴스(Sub-Agent)를 생성**하여 특정 작업을 위임합니다.

일반 도구(Read, Bash, WebFetch 등)가 단일 작업을 수행한다면, Agent 도구는 **여러 도구를 조합하여 복잡한 목표를 스스로 달성**하는 미니 Claude입니다.

## 언제 Agent 도구를 쓰나?

Claude Code는 다음 상황에서 Agent 도구를 자동으로 선택합니다:

- **넓은 범위의 탐색** — 수십 개 파일을 분석해야 할 때
- **독립적인 리서치** — 외부 정보를 수집하고 정리할 때
- **메인 컨텍스트 보호** — 중간 결과물이 많아 메인 대화가 오염될 수 있을 때

직접 지시할 수도 있습니다:
```
"별도 에이전트로 이 작업을 수행해줘"
"메인 컨텍스트에 영향을 주지 않고 분석해줘"
```

## Agent 도구의 동작 흐름

```
메인 Claude
    │
    └── Agent 도구 호출
            │
            ▼
        Sub-Agent (독립 인스턴스)
            ├── WebFetch로 페이지 읽기
            ├── 내용 분석 및 정리
            └── 결과 반환
                    │
                    ▼
        메인 Claude가 결과를 받아 후속 작업
```

Sub-Agent는 **자신만의 컨텍스트 윈도우**에서 작동하므로, 방대한 작업도 메인 대화를 어지럽히지 않습니다.

## 실습: 외부 서비스 지식을 Agents.md로 만들기

Agent 도구의 가장 유용한 활용법 중 하나는 **외부 서비스를 리서치하여 팀 내부 문서로 정리**하는 것입니다.

### 실습 목표

[mop.co.kr](https://support.mop.co.kr/introduce) 서비스 소개 페이지를 Claude가 직접 읽고 분석하여, 팀원 누구나 이해할 수 있는 지식 문서를 작성합니다.

### 프롬프트 예시

```
https://support.mop.co.kr/introduce 페이지를 읽고
mop 서비스가 무엇인지 팀 내부 문서로 정리해줘.
다음 항목을 포함해:
- 서비스 개요 (한 줄 요약)
- 핵심 기능 목록
- 요금제 종류
- 도입 절차
- 주요 개념 용어 정리
```

### 실습 결과물 예시

아래는 위 프롬프트로 Claude가 생성한 문서입니다:

---

#### mop 서비스 개요

**mop**은 LG CNS가 제공하는 **AI 기반 검색광고·쇼핑광고 자동 최적화 플랫폼**입니다. 광고주와 대행사가 캠페인 성과를 높이기 위해 수동으로 하던 입찰 조정, 예산 분배, 이상 감지 등을 자동화합니다.

#### 요금제

| 플랜 | 대상 | 특징 |
|---|---|---|
| Basic | 누구나 | 무료, 회원가입 즉시 사용 |
| Lite | 소규모 광고주 | 2024년 출시 예정 |
| Pro | 중대형 광고주 | 월 구독, 고급 최적화 알고리즘 |
| API Center | 대형 고객사 | 기존 시스템과 직접 연동 |

#### 핵심 기능

- **쇼핑검색 광고 최적화** — 캠페인 관리, 목표입찰, 품절 감지, 경쟁 심화도 모니터링
- **검색광고 최적화** — 자동 입찰 조정, 순위 유지, 예산 관리
- **리포트 & 분석** — 캠페인 리포트, 최적화 리포트, 대시보드
- **예산 최적화** — Spend Pacing(예산 추세 분석), 자동 예산 분배

#### 도입 절차

**Basic:**
```
회원가입 → 비즈니스 유형 입력 → 즉시 사용
```

**Pro / API Center:**
```
도입 사전 설문 → 컨설턴트 매칭 → 맞춤 온보딩 → 월 구독 체결
```

#### 주요 개념 용어

| 용어 | 설명 |
|---|---|
| 애드써클 | mop에서 광고 캠페인을 관리하는 기본 단위 |
| Spend Pacing | 예산 소진 속도를 추적하는 예산 추세 분석 기능 |
| 목표입찰 | 목표 성과(ROAS 등)를 설정하면 자동으로 입찰가를 조정하는 기능 |
| 이상 감지 | 품절, 예산 초과 등 비정상 상황을 자동으로 감지하여 알림 |

## 핵심 정리

Agent 도구 = **"이 작업, 네가 알아서 해줘"**

복잡한 리서치, 대규모 코드베이스 탐색, 외부 서비스 분석 — 이런 작업을 Claude에게 위임할 때 Agent 도구가 내부적으로 동작합니다. 직접 지시하지 않아도 Claude가 적절한 상황에서 자동으로 선택합니다.
