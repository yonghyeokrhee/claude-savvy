# Human-in-the-Loop Workflow

## 개념

긴 Workflow를 실행하다 보면 Claude가 중간에 잘못된 방향으로 가는 경우가 있습니다. **Human-in-the-Loop**는 핵심 결정 지점에 사람의 확인을 넣어 이 문제를 방지하는 패턴입니다.

Claude Code는 `AskUserQuestion` 도구를 통해 작업 중 언제든 사용자에게 질문하고 답변을 기다릴 수 있습니다.

## AskUserQuestion 이란?

Claude Code가 내부적으로 사용하는 도구로, 작업을 **일시 중단**하고 사용자에게 질문을 던집니다. 사용자가 답변하면 그 내용을 바탕으로 작업을 이어갑니다.

```
Claude 작업 실행
    ↓
중요한 결정 지점 도달
    ↓
AskUserQuestion 호출 → 사용자에게 질문
    ↓
사용자 답변 수신
    ↓
답변 기반으로 작업 계속
```

프롬프트에서 직접 유도할 수 있습니다:

```
"각 단계 시작 전에 계획을 먼저 알려주고, 내가 확인하면 진행해줘."
"중간에 불확실한 부분이 있으면 진행하지 말고 먼저 물어봐."
```

## 실습 예제 — MOP 광고 최적화 Workflow

`practice/` 폴더의 MOP 데이터를 활용해 Human-in-the-Loop Workflow를 직접 실행합니다.

### 실습 목표

사용자 확인 없이 자동으로 쭉 실행하는 방식 대신, **단계마다 방향을 확인받는 Workflow**를 경험합니다.

### 프롬프트

```
practice/ 폴더의 데이터를 분석해서 MOP 광고 최적화 액션 아이템을 도출해줘.

단, 다음 규칙을 반드시 지켜:

1. **1단계 완료 후 멈춰**: 데이터를 파악하면 어떤 분석을 진행할지 계획을 먼저 알려줘.
   내가 승인하면 2단계로 넘어가.

2. **2단계 완료 후 멈춰**: 분석 결과를 요약해서 보여줘.
   내가 "계속"이라고 하면 최종 액션 아이템을 작성해.

3. 중간에 데이터가 부족하거나 불확실하면 가정하지 말고 바로 물어봐.
```

### 실행 흐름

```
[1단계] 데이터 파악
  Claude: data/ 폴더 파일 구조 파악
         → "다음과 같이 분석하려 합니다. 진행할까요?"
  사용자: "네, 진행해주세요" / "검색광고 부분은 빼줘"
         ↓ (사용자 답변 반영)

[2단계] 분석 실행
  Claude: 승인된 항목만 분석
         → "분석 결과입니다. 액션 아이템 작성할까요?"
  사용자: "ROAS 기준 정렬해서 작성해줘"
         ↓

[3단계] 액션 아이템 작성
  Claude: 사용자 피드백이 반영된 최종 결과 작성
```

### 확인 포인트가 없을 때 vs 있을 때

| 구분 | 확인 포인트 없음 | Human-in-the-Loop |
|---|---|---|
| 속도 | 빠름 | 느림 |
| 방향 수정 | 끝나고 나서야 가능 | 중간에 즉시 수정 |
| 실수 비용 | 전체 재실행 | 해당 단계부터 재실행 |
| 적합한 상황 | 결과가 예측 가능한 단순 작업 | 중요하거나 결과가 불확실한 작업 |

## Workflow 설계 팁

### 확인 포인트를 넣어야 할 순간

- **방향 결정이 필요한 분기점** — "A방식과 B방식 중 어느 걸로 할까요?"
- **비가역적인 작업 직전** — 파일 덮어쓰기, 외부 API 호출, DB 수정
- **불확실한 데이터** — "이 수치가 맞는지 확인이 필요합니다"
- **긴 작업의 중간 결과 검토** — "여기까지 맞게 가고 있는지 확인해주세요"

### 확인 포인트 없어도 되는 순간

- 읽기 전용 탐색 (파일 구조 파악, 코드 분석)
- 명확하게 정의된 반복 작업
- 결과를 쉽게 되돌릴 수 있는 작업

### 프롬프트 패턴

```
# 패턴 1 — 계획 먼저 확인
"작업 시작 전에 전체 계획을 먼저 보여주고, 내가 OK하면 실행해줘."

# 패턴 2 — 단계별 게이트
"각 단계가 끝나면 다음 단계 진행 여부를 물어봐."

# 패턴 3 — 불확실 시 중단
"확실하지 않은 부분은 가정하지 말고 즉시 질문해줘."

# 패턴 4 — 결정 위임
"선택지가 있으면 내가 고를 수 있게 옵션을 제시해줘."
```
