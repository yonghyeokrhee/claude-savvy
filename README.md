# Claude Code Guide

A practical guide to using Claude Code — Anthropic's official CLI for AI-assisted software development.

## What is Claude Code?

Claude Code is a command-line tool that brings Claude directly into your terminal and codebase. It can read files, write code, run commands, search your project, and help you ship faster.
