# Getting Started

## Install

```bash
npm install -g @anthropic-ai/claude-code
```

## Authenticate

On first run, Claude Code opens the Anthropic login flow:

```bash
claude
```

## Start a Session

```bash
cd my-project
claude
```

You can also run one-line prompts:

```bash
claude "Explain what this project does"
```

## Key Commands

| Command | Description |
|---|---|
| `claude` | Start an interactive session |
| `/help` | In-session help |
| `/clear` | Reset context |
| `/cost` | Check token usage |
| `/exit` | End session |
